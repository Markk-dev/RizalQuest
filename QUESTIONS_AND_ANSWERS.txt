HOW TO ADD CUSTOM QUESTIONS
================================================================================

To add custom questions for any chapter:

1. Open lib/questions.ts
2. Add a new chapter number to QUESTION_BANK (e.g., "3: {")
3. Add levels 1-5 with 5 questions each
4. Use the same format as Chapters 1 and 2

Example structure:
3: {
  1: [
    { type: "multiple-choice", question: "...", options: [...], correct: 0 },
    { type: "typing", question: "...", correctAnswer: "..." },
    // ... 3 more questions
  ],
  2: [ /* 5 questions */ ],
  3: [ /* 5 questions */ ],
  4: [ /* 5 questions */ ],
  5: [ /* 5 questions */ ],
}

Available question types:
- multiple-choice
- typing
- multi-input-typing
- matching
- classification
- word-arrangement
- fill-in-blanks
- drawing
