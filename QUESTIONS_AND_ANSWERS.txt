NOTES FOR EDITING:
================================================================================

1. Multiple Choice Questions:
   - The correct answer is marked with ✓
   - Options are labeled A, B, C, D
   - The "correct" field in the code uses index (0=A, 1=B, 2=C, 3=D)

2. Typing Questions:
   - Answer is case-insensitive in the code
   - Keep answers simple (one word preferred)

3. Word Arrangement:
   - Words will be shuffled when displayed
   - Correct order must match exactly

4. Classification:
   - Items will be dragged to categories
   - Each item must map to exactly one category

5. Fill in the Blanks:
   - Two blanks per question
   - Order matters (first blank, second blank)

6. Matching:
   - Pairs of left and right items
   - Used for connecting related concepts
